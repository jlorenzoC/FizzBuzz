FizzBuzz
Which number is Fizz, Buzz, or FizzBuzz?

FizzBuzz is an algorithm that processes the numbers from 1 to 100. It will print the word "Fizz", whenever 3 and its multiples are encountered, "Buzz" whenever 5 and its multiples are encountered, and "FizzBuzz", whenever numbers that are multiples of both 3 and 5, are encountered, in another case, it prints the number.
